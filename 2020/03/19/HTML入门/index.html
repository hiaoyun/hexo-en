sample

<html>
    <head>
        <meta http-equiv="Content-Type" content="text/html; charset=utf-8">
        <title>制作我的第一个网页</title>
    </head>
    <body>
        <h1>Hello HTML</h1>

HTML+CSS+JavaScript

HTML（Hyper Text Markup Language）是网页内容的载体。

CSS（Cascading Style Sheet）是表现。

JavaScript用来实现特殊效果。

标签

不区分大小写，建议小写。

  • 标题标签

      <h1>
     标题标签 
  </h1>

  • 段落标签

      <p>
      段落标签
  </p>

  • img标签

      <img src="test.jpg">

  • 根标签

    <html></html>

  • <head></head>头部元素的容器。包含文章的属性和信息

    头部元素：

    • <title>网页的标题，出现在标题栏
    • <script>
    • <style>
    • <link>
    • <meta>

  • <body></body>网页内容的容器

    网页内容：

    • <h1> 标题 <h1> <h2> <h3> <h4> <h5> <h6>
    • <p>段落 paragragh
    • <a>
      • 属性：
        • href
        • title
        • target _blank表示在新标签页打开，省略该属性即在当前页打开
    • <img>

  • <strong>粗体

  • <em>斜体

  • <span>

    为文字设置单独样式

      <style>
      span{
          color:blue;
      }    
  </style>

  • <q>引用较短文字，引用文本不用加引号

  • <blockquote>长文本引用

  • <br/>换行 (xhtml1.0) <br>(html4.01)

  • &nbsp;空格，字符实体

  • <hr/>水平分割线

  • <address>地址标签，显示的样式为斜体

  • <code>单行代码

  • <pre>多行代码

  • <ul>

  • <ol>

  • <div>

      <div id="标识">

  </div>

  • <table>

    • <tr>一行
    • <th>表格表头
    • <td>单元格

语义化

不同内容放在对应的标签中。

用<h1></h1>来表示标题，而不要用加粗加大的字体来表示。
