新建脚本

vim cnt.sh

逐行显示指定文件的每一行内容并统计总行数

编写脚本

#!/bin/bash
# cnt.sh
# 逐行显示指定文件的每一行内容并统计总行数

# 总行数计数器
cnt=0

# 读取一行
while read line
do
    # 输出这行内容
    echo $line
    # 计数器加一
    cnt=$(expr $cnt + 1)
# file为指定文件
done < file

# 输出总行数
echo $cnt

指定测试文件file内容

Hello
,
World
!

运行脚本

bash cnt.sh
